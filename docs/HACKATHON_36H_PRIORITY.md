# TenantShield — приоритизация на 36 часов хакатона

Цель: **работающее демо**, которое закрывает треки AWS Agentic AI, Abelian Privacy & AI, HKUST Innovation и Main Awards.

---

## Порядок сборки (рекомендуемый)

### Фаза 1: Фундамент (≈6–8 ч)

| # | Задача | Зачем |
|---|--------|-------|
| 1 | Next.js + TS + Tailwind, адаптивная оболочка | Все экраны работают на телефоне и десктопе |
| 2 | Landing: слоган, 6 модулей (карточки), CTA «Try demo» | Первое впечатление и навигация |
| 3 | Подключение QDay testnet (wagmi/viem или ethers + MetaMask) | **Must** для Abelian Award; один экран «Connect Wallet» |
| 4 | .env: `NEXT_PUBLIC_QDAY_RPC`, `AWS_REGION`, Bedrock agent ARNs | Чтобы не блокироваться на конфиге позже |

### Фаза 2: Ядро демо (≈12–14 ч)

| # | Задача | Зачем |
|---|--------|-------|
| 5 | **Contract Analyzer:** upload PDF → API route → Bedrock (или прямой InvokeModel) → отчёт на экран | AWS Agentic AI + демо «AI без личности» |
| 6 | **Deposit Pool (упрощённый):** смарт-контракт Escrow на QDay (deposit / release), UI: «Внести депозит» → tx → успех | Abelian Award: техническое исполнение и приватность |
| 7 | Страница «Мои инструменты»: ссылки на Analyzer и Deposit | Связность демо |

### Фаза 3: Второй агент и «коллектив» (≈8–10 ч)

| # | Задача | Зачем |
|---|--------|-------|
| 8 | **Collective Pool (минимально):** форма «район + бюджет + дата» → Bedrock Agent 2 агрегирует и возвращает текст «коллективное предложение» | Второй агент для AWS трека; анонимность через «только агрегат» |
| 9 | **Legal Fund (статика):** экран «$5/мес в смарт-контракт», кнопка «Внести» → tx на QDay | Показать ещё один use case Abelian |
| 10 | **Reviews (статика или один отзыв):** «Оставить отзыв» → хэш/commit в контракт без раскрытия автора | Privacy creativity для жюри |

### Фаза 4: Полировка и pitch (≈4–6 ч)

| # | Задача | Зачем |
|---|--------|-------|
| 11 | Слайд «Архитектура: данные → Abelian → AWS» + скриншоты демо | Abelian и Main Awards |
| 12 | Короткое видео (2–3 мин): сценарий «студент проверяет договор → вносит депозит в escrow → видит коллективное предложение» | Impact и novelty |
| 13 | README: как запустить, как подключить QDay, какие env нужны | Судейство и воспроизводимость |

## Риски и запасные варианты

| Риск | Запасной вариант |
|------|-------------------|
| Bedrock Agent не успевают настроить | Один агент или прямой InvokeModel (Claude) с промптом «ты юрист по HK tenant law» |
| QDay testnet нестабилен | Документировать RPC и показать контракты в коде; при необходимости коротко показать Sepolia + тот же Solidity |
| Нет времени на второй агент | Один агент (Contract Analyzer + простой «aggregator» промпт в том же потоке) с чётким разделением в документации на «Agent 1» и «Agent 2» |
